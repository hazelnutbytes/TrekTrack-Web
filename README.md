# TrekTrack – Guided Expedition Booking

## Background
Planning mountain treks can sometimes be confusing for beginners. Many trekking websites only show basic information and don't clearly explain things like the full itinerary, altitude changes, required gear, or the total cost when renting equipment. Because of this, users often struggle to understand what they need before booking a trek.

## Abstract
TrekTrack is a front-end prototype of a guided trekking booking platform. The goal of this project is to demonstrate how trekking companies could present important information in a clear and interactive way. The website shows details such as trek itineraries, elevation profiles, packing lists, and gear rental options.

The project is built using **HTML, CSS, and JavaScript**, using the Kedarkantha Trek as an example trek to demonstrate the layout and features. Users can explore the itinerary, check trek difficulty information, view elevation changes, and select optional trekking gear. When gear is selected, the **total price updates dynamically using JavaScript**, simulating a real booking experience.

This project focuses mainly on **UI design, layout structure, and basic interactivity** rather than a full booking system with backend integration.

## Features
- Trek itinerary timeline
- Elevation profile visualization
- Packing list for trekkers
- FAQ section
- Gear rental selection
- Dynamic total price update using JavaScript
- Outdoor-inspired UI design
